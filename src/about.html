What is mirror?


Mirror is a site where you can remotely access your SQL-based
databases. Mirror gives you the power to use SQL querys to view, edit
and manage your databases anywhere.

To get started, enter in your database's details.
Once a connection is established, you can type SQL into the command
box and click run to submit. The response to this will be displayed
below.


Privacy


To get this out the way, this site can't access any of your database credentials.
This being said, MAKE SURE YOUR CONNECTION TO THIS SITE IS SECURE before connecting.
Credentials are posted to our server, if an attacker intercepts this and https
isn't being used, it is possible that they could then access this data unencrypted, so be carefull!
